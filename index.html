Projetos em Ciência de Dados

Soluções para o Mercado Imobiliário

O projeto foi desenvolvido para solucionar desafios enfrentados por uma imobiliária no Rio de Janeiro, aplicando técnicas avançadas de ciência de dados para melhorar a precificação de imóveis e recomendações personalizadas aos clientes. Com uma abordagem inovadora, o projeto integrou análise de dados, modelos de regressão e algoritmos de recomendação, oferecendo insights valiosos e soluções práticas para otimizar a operação e estratégia da imobiliária.

As etapas do projeto incluíram:

  • Análise e Tratamento de Dados: Estruturação de dados relacionados a preços, características e localização dos imóveis.
  • Modelo de Regressão: Criação e ajuste de um modelo Random Forest Regressor para precificação precisa, avaliado com métricas como R² e RMSE.
  • Sistema de Recomendação: Aplicação de PCA para redução de dimensionalidade e desenvolvimento de recomendações personalizadas com clusterização e distância euclidiana.

As ferramentas utilizadas foram:

  • Python, Estatística, Análise de Dados, Machine Learning.
  • PySpark, Pandas, NumPy, Sklearn.
  • Random Forest Regressor, PCA, Clusterização e Distância Euclidiana.
  • Limpeza de Dados, Normalização, Métricas de Avaliação (R², RMSE).

E-commerce Solutions

O projeto E-commerce Solutions forneceu insights valiosos e soluções práticas para os desafios enfrentados pela ShopGlobal, melhorando significativamente a operação e a estratégia da empresa através da aplicação de ciência de dados.

Projeto de Previsão de Inadimplência de Empréstimos

Envolve a análise de dados de empréstimos e o desenvolvimento de um modelo de aprendizado de máquina para prever inadimplências, permitindo que a WBK tome medidas proativas para reduzir a taxa de inadimplência. O resultado final do projeto foi um modelo com uma precisão de 79%, o modelo pode melhorar significativamente a capacidade da WBK de prever inadimplências. Ao identificar clientes de alto risco antecipadamente, a WBK pode implementar medidas para mitigar o risco, potencialmente reduzindo a taxa de inadimplência de 35% para abaixo de 25%.

As ferramentas utilizadas foram:

  • Python, Estatística, Análise de Dados, Machine Learning.
